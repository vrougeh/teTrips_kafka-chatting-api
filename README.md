## teTrips (together edit Trips)

tetrips의 Kafka를 활용한 채팅 기능 및 Kubernetes와 Jenkins를 사용한 CI/CD 를 위한 레포지토리 입니다.

SpringBoot3를 활용하여 구현하였습니다.



#### 👨‍🏫프로젝트 소개

다수의 사용자가 채팅으로 소통하며, 실시간 동시 편집 기능을 활용하여 지도에서 여행 동선을 계획하는 웹 사이트 입니다.
